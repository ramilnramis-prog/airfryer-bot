# 50 рецептов для аэрогриля 🔥

Лид-магнит для Telegram. Все рецепты — для приготовления **в силиконовой форме**:
готовите в форме → чаша аэрогриля остаётся чистой → форму ополоснули и всё.

Формат: **Название — ингредиенты; температура / время.**
Температуры усреднённые, ориентируйся на свой аэрогриль (4–7 л).

> 💡 Этот файл — основа контента канала: публикуй по 1–2 рецепта в день (см. контент-план),
> и людям всегда есть зачем заходить и оставаться подписанными.

## Как отдавать лид-магнит в Telegram
1. Закреп в канале: «Подпишись → забери все 50 рецептов» + ссылка на этот пост/файл.
2. Вариант А — выложить файл (PDF) в закреп. Вариант Б — бот выдаёт PDF за подписку.
3. Дальше публикуешь рецепты по 1–2 в день с фото/видео (можно генерить в стиле ролика),
   чтобы канал жил и подписчик не отписывался.
4. В каждом 3–4-м рецепте — мягкое напоминание: «готовлю в силиконовой форме, чаша чистая;
   форма на Ozon, артикул 1931921872».
