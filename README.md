# Telegram-бот воронки «Аэрогриль без мытья»

Автоматизирует Telegram-часть: выдаёт лид-магнит «50 рецептов» новым подписчикам
и сам публикует рецепты в канал по расписанию с периодическим CTA на Ozon.

```
Видео/Reels → бот выдаёт 50 рецептов → подписка → автопостинг рецептов → покупка на Ozon
```

---

## Что делаю я (уже готово в этой папке)
- `bot.py` — логика бота
- `recipes.json` — очередь рецептов (50 шт; дополняй своими из своей книги)
- `lead_magnet.md` — файл «50 рецептов», который бот отдаёт по `/start`
- `requirements.txt`, `Procfile`, `Dockerfile` — для запуска/деплоя

## Что делаешь ты (15 минут, один раз)
Ниже по шагам. Бот без этих данных не заработает — это нельзя автоматизировать за тебя.

### Шаг 1. Создай канал
1. В Telegram → создай **канал** (публичный, с @username, напр. `@aero_recepty`).
2. Запиши ссылку вида `https://t.me/aero_recepty`.

### Шаг 2. Создай бота и получи токен
1. Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → имя и username бота.
2. Скопируй **токен** (вид `123456:ABC...`). ⚠️ Это секрет — никому не показывай, в чат лучше не вставляй.
3. Добавь бота **администратором своего канала** (права: публиковать сообщения).

### Шаг 3. Узнай CHANNEL_ID
- Проще всего: используй `@username` канала (если он публичный) — например `@aero_recepty`.
- Если канал приватный — нужен числовой id вида `-100...` (можно через @userinfobot переслать пост из канала).

### Шаг 4. Деплой в облако (Railway — бесплатно, рекомендую)
1. Залей эту папку `telegram-bot/` в репозиторий на GitHub.
2. Зайди на [railway.app](https://railway.app) → **New Project** → **Deploy from GitHub repo** → выбери репозиторий.
3. Railway сам поставит зависимости (по `requirements.txt`) и запустит `worker` (по `Procfile`).
4. В разделе **Variables** добавь переменные окружения (см. список ниже).
5. Готово — бот работает 24/7, от твоего ПК не зависит.

> Альтернатива — Render.com (тип сервиса **Background Worker**), либо свой VPS через `Dockerfile`.

### Шаг 5. Переменные окружения (вписать на хостинге)
| Переменная | Что это | Пример |
|------------|---------|--------|
| `BOT_TOKEN` | токен от BotFather | `123456:ABC...` |
| `CHANNEL_ID` | канал для постинга | `@aero_recepty` |
| `OZON_ARTIKUL` | артикул товара | `1931921872` |
| `OZON_LINK` | ссылка на карточку | `https://www.ozon.ru/product/...` |
| `TG_CHANNEL_LINK` | ссылка на канал (для кнопки) | `https://t.me/aero_recepty` |
| `POST_TIMES` | время постинга | `10:00` или `10:00,18:00` |
| `TZ` | часовой пояс | `Europe/Moscow` |
| `CTA_EVERY` | CTA на Ozon каждый N-й пост | `4` |

### Шаг 6. Закрепи лид-магнит в канале
- Открой бота → `/start` → он пришлёт файл `50-receptov-aerogril.md`.
- (Опционально) преобразуй его в PDF и **закрепи** в канале постом «Подпишись → забери 50 рецептов».

## Как добавить свои рецепты
Открой `recipes.json` и добавь объекты в конец списка:
```json
{"category": "Курица", "title": "Название", "ingredients": "что нужно", "mode": "180°C / 15 мин"}
```
Категории с эмодзи: Курица, Мясо, Рыба, Овощи, Выпечка, Завтраки, Снеки, Десерты.
Бот постит по порядку и идёт по кругу, когда список закончится.

## Локальный тест (если выбрал «Мой ПК»)
> На твоём ПК сейчас нет рабочего Python — сначала поставь Python 3.12 с python.org.

```bash
pip install -r requirements.txt
# задать переменные окружения (или создать .env и экспортировать)
python bot.py
```

## Безопасность
- `BOT_TOKEN` — секрет. Храни только в переменных окружения хостинга. `.env` и `state.json` уже в `.gitignore`.
- Если токен где-то засветился — в @BotFather сделай `/revoke` и получи новый.
